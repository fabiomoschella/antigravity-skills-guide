## Contributing and Attribution

### How to Contribute Skills

1. **Fork** the Antigravity repository
2. **Create** your skill following the standard format
3. **Add attribution** in the skill metadata
4. **Submit** a pull request

### Attribution Format for New Skills

When contributing a new skill, include this metadata:

```yaml
---
name: skill-name
description: Brief description
author: Your Name
source: original | derived | curated
based_on: [if derived, reference base skill]
license: MIT
version: 1.0.0
---
```

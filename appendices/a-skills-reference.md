## Skill Selection Guide

### By Task Type

**Starting a new project:**
- `@architecture` for overall design
- `@tech-stack-selector` for technology choices
- `@project-planner` for timeline

**Debugging issues:**
- `@debugger-expert` for methodology
- `@production-debugger` for live systems
- `@memory-debugger` for memory issues

**Performance optimization:**
- `@performance-tester` for testing
- `@database-optimization` for queries
- `@finops-analyst` for cost

**Security review:**
- `@appsec-expert` for application security
- `@secure-code-reviewer` for code review
- `@threat-hunter` for threats

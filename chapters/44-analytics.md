# Chapter 44: Analytics & Reporting

**Last Updated:** February 5, 2026

---

## Overview

Analytics and reporting transform data into actionable insights. This chapter covers essential skills for building data visualizations and business intelligence solutions.

### Skills Covered

| Skill | Purpose | Best For |
|-------|---------|----------|
| `@analytics-engineer` | Analytics pipelines | Data modeling |
| `@bi-developer` | Business intelligence | Dashboards, reports |
| `@visualization-expert` | Data visualization | Charts, graphs |

---

## 44.1 Analytics Engineering with @analytics-engineer

```text
@analytics-engineer Design analytics data model:

Requirements:
- Product analytics for SaaS
- User behavior tracking
- Funnel analysis
- Cohort analysis

Provide:
1. Data modeling approach
2. Event tracking design
3. Aggregation strategy
4. Query optimization
```

---

## 44.2 Business Intelligence with @bi-developer

```text
@bi-developer Create executive dashboard:

Metrics needed:
- Revenue and growth
- Customer acquisition
- Product usage
- Operational efficiency

Provide:
1. Dashboard design
2. Tool selection (Looker, Tableau, etc.)
3. Data refresh strategy
4. Access control
```

---

## 44.3 Data Visualization with @visualization-expert

```text
@visualization-expert Design data visualization strategy:

Context: Customer-facing analytics

Requirements:
- Embedded analytics
- Real-time updates
- Mobile responsive
- White-label support

Provide:
1. Visualization library selection
2. Chart type recommendations
3. Interactivity patterns
4. Performance optimization
```

---

## Best Practices

1. **Tell a Story:** Data should narrate
2. **Keep It Simple:** Avoid chart junk
3. **Performance:** Optimize for speed
4. **Accessibility:** Color-blind friendly
